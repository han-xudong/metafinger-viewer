# Meta-Finger Viewer

This is a viewer example for the [Meta-Finger](https://github.com/han-xudong/meta-finger).

## Quick Start

Clone the latest repository and install the dependencies:

```bash
git clone https://github.com/han-xudong/meta-finger-viewer.git
cd meta-finger-viewer
pip install -r requirements.txt
```

Run the viewer:

```bash
python server.py
```
